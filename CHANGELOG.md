# 更新日志

## [1.1.0] - 2026-05-11

### 新增
- **WebSocket 实时通知系统**：支持连接管理、心跳检测、广播通知、自动重连
- **Service Worker PWA 离线支持**：静态资源缓存、API 离线降级、后台同步、推送通知
- **数据库备份/恢复功能**：JSON 格式备份、CSV 导出、恢复操作、备份历史记录
- **性能监控中间件**：请求响应时间统计、慢查询检测、端点性能分析、内存/数据库监控
- **前端缓存策略**：LocalStorage 缓存、SessionCache、后台数据同步、离线突变队列
- Flyway V4 迁移脚本（backup_logs 表）

### 优化
- 健康检查端点增加 WebSocket 连接数和服务器运行时间
- 侧边栏导航新增数据备份入口
- 项目自检脚本扩展至 105+ 文件检查

## [1.0.0] - 2026-05-11

### 新增
- 完整的用户认证系统（JWT + bcrypt）
- 学生信息管理（CRUD、搜索、筛选、分页）
- 成绩管理系统（录入、统计、分析）
- 数据看板（统计卡片、图表展示）
- AI助手（自然语言查询数据库）
- 课程管理、班级管理、操作日志
- 数据导入/导出（CSV格式）、统计报表
- 响应式布局设计、Docker Compose部署
- Flyway数据库迁移、健康检查与监控
- 速率限制、请求日志记录、测试套件（Jest）

### 技术栈
- 后端：Node.js + Express + PostgreSQL
- 前端：原生HTML/CSS/JS + Canvas图表
- AI：OpenAI-compatible API
- 部署：Docker + Docker Compose
