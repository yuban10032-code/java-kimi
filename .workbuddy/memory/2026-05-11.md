## 第二轮迭代 - 系统增强 (v1.1.0)

在已完成的学生管理系统基础上，新增了6大功能模块：

1. **WebSocket 实时通知系统** (`backend/websocket.js`)
   - Express HTTP 服务器集成 ws 模块
   - 客户端连接管理、心跳检测（30s间隔）、自动清理
   - 广播/定向通知 API，前端自动重连机制（指数退避）

2. **Service Worker PWA 离线支持** (`frontend/service-worker.js`)
   - 静态资源 Cache-first 策略、API Network-first 策略
   - 离线时返回 index.html 兜底、IndexedDB 后台同步
   - Push 通知支持、manifest.json 已存在

3. **数据库备份/恢复功能** (`backend/routes/backup.js`, `frontend/backup.html`)
   - JSON 全量备份、CSV 按表导出、恢复操作（事务保护）
   - Flyway V4 迁移添加 backup_logs 表
   - 前端备份历史列表、一键下载备份文件

4. **性能监控中间件** (`backend/middleware/performance.js`)
   - 请求响应时间追踪、慢查询检测（>500ms）
   - 端点级统计、小时级聚合、错误追踪
   - 数据库连接数/表大小/索引状态查询

5. **前端缓存策略** (`frontend/js/cache.js`)
   - LocalStorage TTL 缓存、SessionCache、DataSync 后台刷新
   - OfflineQueue 离线突变队列

6. **文档与自检更新**
   - CHANGELOG.md 更新至 v1.1.0
   - README.md 功能列表/项目结构同步更新
   - check-project.js 扩展至 111 项检查，全部通过
