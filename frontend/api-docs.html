API文档

接口基础信息

Base URL: http://localhost:3001/api

认证方式: Bearer Token (JWT)

请求头: Authorization: Bearer <token>

认证

POST /auth/login
用户登录，返回JWT令牌
POST /auth/register
用户注册

学生管理

GET /students
获取学生列表（支持分页、搜索、筛选）

查询参数:

参数类型说明
pageinteger页码，默认1
limitinteger每页数量，默认10
searchstring搜索关键词（学号/姓名/电话）
majorstring按专业筛选
class_namestring按班级筛选
gradestring按年级筛选
statusstring按状态筛选
GET /students/:id
获取学生详情（包含成绩）
POST /students
创建学生
PUT /students/:id
更新学生信息
DELETE /students/:id
删除学生

成绩管理

GET /scores
获取成绩列表
GET /scores/student/:id
获取指定学生的成绩统计
POST /scores
录入成绩

数据看板

GET /dashboard/stats
获取统计数据（学生数、课程数、平均分、及格率等）
GET /dashboard/charts
获取图表数据（成绩分布、学期趋势、TOP10等）

AI助手

POST /agent/query
自然语言查询数据库（AI生成SQL并执行）

导出

GET /export/students
导出学生数据为CSV
GET /export/scores
导出成绩数据为CSV
